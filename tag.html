링크를 참고해도 모르겠는 부분은 부장한테 물어보기! (* 부장도 모를 수 있음... 구글의 도움을 받아 함께 해결합시다) 이해가 잘 안되면 마우스 우클릭 + 소스코드보기로 이 웹페이지의 코드를 볼 수도 있습니다.

  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!
  • tag - html 문법의 핵심, 태그
  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

* 태그를 쓸 때는 기본적으로 여는 태그 + 태그 효과가 적용될 글 + 닫는 태그 순으로 씀

[1] h1~h6 : headline, 제목 - 숫자가 클수록 글씨 크기는 작아짐

[2] strong : 진한 글씨 (ex : 진한 글씨)
[3] u : underline, 밑줄 (ex : 밑줄)
[4] br : 줄바꿈 !닫는 태그 쓰지 않음!
[5] p : 단락 바꿈 !닫는 태그 있음! - 줄바꿈과 별 차이 없어보이지만 단락을 구분할 때는 코드의 가독성을 위해 p 태그를 쓰는 것이 좋음
* 태그 중첩도 가능 (ex : 진한 글씨 + 밑줄)


===목차를 만드는 태그 li, ul, ol===
[6] li : list, 목차를 만듦
[7] ul : unordered list, "li"태그로 쓴 목차 전체를 묶어 목차의 경계를 만듦. 목차 태그계의 "p"태그라고 생각하면 됨.
[8] ol : ordered list, "li"태그로 쓴 목차 전체를 순서 표시와 함께 묶어 목차의 경계를 만듦.

  • 목차의
