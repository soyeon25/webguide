2020 자율동아리 웹프로그래밍

아래 링크를 참고하면서 자유롭게 문서형 웹페이지를 제작해봅시다!
링크를 참고해도 모르겠는 부분은 부장한테 물어보기! (* 부장도 모를 수 있음... 구글의 도움을 받아 함께 해결합시다) 이해가 잘 안되면 마우스 우클릭 + 소스코드보기로 이 웹페이지의 코드를 볼 수도 있습니다.

  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!
  • tag - html 문법의 핵심, 태그
  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

=== 링크 걸기 ===

  1. 프로젝트 파일 안에 링크될 웹페이지를 코딩하기
  2. 처음 코딩하던 초기 웹페이지로 돌아와서, a 태그 열기 + href="링크될 페이지 파일의 이름" + a 태그 닫기 + 웹페이지에 표시될 링크의 이름 형식으로 작성
  3. href 뒤에 페이지 파일 이름 대신 인터넷 사이트 링크를 복사+붙여넣기하면 내 컴퓨터에 있는 파일이 아닌 인터넷 사이트를 링크할 수도 있음
  4. 이렇게!

=== 이미지 삽입은 태그와 다른 개념! 열고 닫는 개념이 없음. <> 안에 필요 정보를 모두 넣기! ===
img : 이미지 삽입

  • 인터넷에 있는 이미지를 삽입하는 방법 : 'img scr=이미지링크' 형식으로 작성
  • 내 컴퓨터에 있는 이미지를 삽입하는 방법 : 이미지 파일을 프로젝트 폴더에 넣고, 'img scr=이미지파일 이름' 형식으로 작성
  • 이미지의 크기를 바꾸는 방법 : 'width=n%' 형식으로 작성
