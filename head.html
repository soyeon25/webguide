2020 자율동아리 웹프로그래밍

아래 링크를 참고하면서 자유롭게 문서형 웹페이지를 제작해봅시다!
링크를 참고해도 모르겠는 부분은 부장한테 물어보기! (* 부장도 모를 수 있음... 구글의 도움을 받아 함께 해결합시다) 이해가 잘 안되면 마우스 우클릭 + 소스코드보기로 이 웹페이지의 코드를 볼 수도 있습니다.

  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!
  • tag - html 문법의 핵심, 태그
  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

* 코드의 head에 들어가야 할 필수 요소

  • !doctype html : 코드 맨 앞에 씀. 이 웹페이지가 html로 만들어졌다는 것을 표시
  • meta charset="utf-8" : 한글 깨짐 방지
  • title : 웹페이지 내용에는 영향을 미치지 않지만, 내용을 반영하는 제목은 필요함!
  • head : 이 항목에 쓰여있는 것들, 즉, 본문에 직접적으로 드러나지는 않지만 본문을 설명하는 정보들을 head 태그로 감싸줘야 함
  • body : head를 제외한 나머지, 즉, 본문을 body 태그로 감싸줘야 함
  • html : html 태그로 head 태그부터 body 태그까지 전부 다 감싸줘야 함
