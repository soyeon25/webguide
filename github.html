2020 자율동아리 웹프로그래밍

아래 링크를 참고하면서 자유롭게 문서형 웹페이지를 제작해봅시다!
링크를 참고해도 모르겠는 부분은 부장한테 물어보기! (* 부장도 모를 수 있음... 구글의 도움을 받아 함께 해결합시다) 이해가 잘 안되면 마우스 우클릭 + 소스코드보기로 이 웹페이지의 코드를 볼 수도 있습니다.

  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!
  • tag - html 문법의 핵심, 태그
  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

우리가 지금 만든 웹페이지는 각자의 컴퓨터에서만 접속할 수 있는 상태입니다.
다른 컴퓨터에서도 접속할 수 있는 진짜 웹사이트로 만들기 위해서 github의 웹호스팅 기능을 이용하겠습니다.
과정이 조금 복잡할 수 있으니 천천히 읽으면서 따라해봅시다!!

  1. sign in을 눌러 github 사이트에 로그인한다.
  2. 우측 상단 아이콘 클릭 - your repositories 클릭
  3. 초록색 'new'아이콘 클릭
  4. Repository name 에는 프로젝트의 이름을 적는다.
  5. public(공개), private(비공개) 중 public을 체크한다.
  6. Initialize this repository with a README을 체크한다.
  7. Create repository(저장소 생성하기) 클릭
  8. add file - uplode files 클릭
  9. choose your files - 프로젝트 폴더 속에 있는 모든 파일선택
  10. 뭔가 쓸 수 있는 말풍선 칸이 있는데, 여기에는 웹페이지에 대한 설명을 적으면 됨
  11. Commit changes 클릭 - 아까 봤던 화면으로 되돌아오면 settings 클릭
  12. 쭉 내리다보면 GitHub Pages가 나옴. 여기서 source를 master branch로 설정!
  13. 그러면 새로고침되는데, 다시 GitHub Pages로 쭉 내려보면 웹페이지 주소가 나와있음!
  14. 깃허브에서 준 주소 끝에 /기초웹페이지 파일이름 을 붙인 것이 우리의 웹페이지 주소가 됨. 어떤 기기로든 웹페이지 접속 가능!!
