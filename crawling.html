  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!

  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

크롤링에는 Python을 사용합니다
크롤링을 위한 기본 코드는 다음과 같습니다

#requests, BeautifulSoup, re 세 가지의 모듈을 사용합니다
import requests
from bs4 import BeautifulSoup
import re
#원하는 웹페이지와 웹페이지의 소스코드를 불러옵니다
변수명1 = requests.get("웹페이지 링크")
변수명2 = BeautifulSoup(변수명1.content, "html.parser")
#원하는 데이터를 뽑아냅니다
변수명3 = str(변수명2.find_all(attrs={'요소명':'요소 이름'}))
#뽑아낸 데이터에서 태그를 지우고 텍스트만 남깁니다
print(re.sub('<.+?>', '', 변수명3, 0).strip())
