2020 자율동아리 웹프로그래밍

아래 링크를 참고하면서 자유롭게 문서형 웹페이지를 제작해봅시다!
링크를 참고해도 모르겠는 부분은 부장한테 물어보기! (* 부장도 모를 수 있음... 구글의 도움을 받아 함께 해결합시다) 이해가 잘 안되면 마우스 우클릭 + 소스코드보기로 이 웹페이지의 코드를 볼 수도 있습니다.

  • 개발환경 - 웹페이지를 만들기 전에
  • head - html 코드를 짤 때 맨 앞에 꼭 적어야 하는 요소들!
  • tag - html 문법의 핵심, 태그
  • image & link - 이미지 삽입과 링크 삽입
  • 웹서버 구축 - github에서 웹서버 구축하기
  • 크롤링 - python을 이용한 크롤링, python 결과값을 html 웹페이지에 나타내기

06/28 오늘은 atom과 github만 사용합니다!

  1. 앞으로 웹프로그래밍을 할 때 사용할 폴더를 하나 만든다.
  2. atom을 실행시킨다.
  3. 좌측 상단 file - add project folder - 아까 만들었던 폴더 선택
  4. 왼쪽에 표시되는 폴더를 우측클릭하고 New file 클릭 - 파일이름.html 으로 이름 정하기
  5. control+O를 누르고 해당 파일로 들어가면 내 컴퓨터에서 해당 웹페이지를 실행시킬 수 있음!
